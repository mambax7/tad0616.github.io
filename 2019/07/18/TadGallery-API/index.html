TadGallery API

這是寫給 XOOPS 的 tadgallery 相簿模組用的API，json格式。

取得所有分類

http://網域/modules/tadgallery/app_api.php?op=get_cates
參數：

  • get_cates

https://www.lces.tn.edu.tw/modules/tadgallery/app_api.php?op=get_cates 回傳內容：

  • csn：分類編號
  • title：分類標題
  • of_csn：父分類編號
  • sort：分類排序
  • cover：相簿代表圖
  • url：相簿的網址
