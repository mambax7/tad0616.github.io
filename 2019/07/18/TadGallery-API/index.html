TadGallery API

2019-07-18
XOOPS, api, tadgallery, 電子相簿模組

這是寫給 XOOPS 的 tadgallery 相簿模組用的API，回傳json格式。

取得所有相簿

http://網域/modules/tadgallery/app_api.php?op=get_cates&of_csn=父相簿（分類）編號
參數：

  • op = get_cates
  • of_csn = 0

https://www.lces.tn.edu.tw/modules/tadgallery/app_api.php?op=get_cates&of_csn=0 回傳內容：

  • csn：相簿編號
  • title：相簿標題
  • of_csn：父相簿編號
  • dir_count：子相簿數量
  • file_count：相片數量
  • sort：相簿排序
  • cover：相簿代表圖
  • url：相簿的網址

取得某相簿照片

http://網域/modules/tadgallery/app_api.php?op=list_all_photos&csn=相簿編號
參數：

  • op：list_all_photos
  • csn：相簿編號

https://www.lces.tn.edu.tw/modules/tadgallery/app_api.php?op=list_all_photos&csn=3 回傳內容：

  • sn：相片編號
  • csn：相簿編號
  • title：相片標題
  • post_date：發布日期
  • description：相片說明
  • filename：相片檔名
  • counter：閱讀人次
  • tag：標籤
  • photo_sort：排序
  • photo_l：大圖連結
  • photo_m：中圖連結
  • photo_s：小圖連結
  • album_title：相簿標題
  • is360：是否為360相片
