取得網站資訊

http://網域/modules/tadtools/app_api.php?op=web_info
參數：

  • web_info

回傳內容：

  • sitename：網站名稱
  • slogan：網站口號
  • adminmail：管理員信箱
  • meta_description：網站簡介
  • slide：網站圖片
