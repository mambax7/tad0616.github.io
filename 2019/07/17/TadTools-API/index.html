TadTools API

XOOPS api tadtools 網站工具包模組 2019/07/17

取得網站資訊

http://網域/modules/tadtools/app_api.php?op=web_info
參數：

  • web_info

http://www.lces.tn.edu.tw/modules/tadtools/app_api.php?op=web_info 回傳內容：

  • sitename：網站名稱
  • slogan：網站口號
  • adminmail：管理員信箱
  • meta_description：網站簡介
  • slide：網站圖片

取得該站有使用的模組

http://網域/modules/tadtools/app_api.php?op=web_modules
參數：

  • web_modules

http://www.lces.tn.edu.tw/modules/tadtools/app_api.php?op=web_modules 回傳內容：

  • name：模組名稱
  • dirname：目錄名稱
