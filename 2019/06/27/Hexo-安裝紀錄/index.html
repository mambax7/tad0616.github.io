Hexo 安裝紀錄

2019-06-27
hexo, 安裝

hexo 是一套靜態頁面產生器，之前沒玩過這類型的工具，恰巧需要一個簡單的部落格，也想嘗試直接架在 gihub.io 上，於是簡單紀錄一下其安裝方式…
是說…我對 markdown 很不熟阿！

一、安裝hexo

  • 先安裝node.js、git

  • 建立個資料夾，如 hexo

  • 進入資料夾，開啟終端機，初始化專案

  • -f表示force的意思，不加這個，npm會讓你輸入一堆信息，比如name、version之類，如果只是做做實驗小demo，直接-f，npm幫你初始化package.json，並填充各種默認值，省事。

  • 安裝 Hexo

  • 用hexo建立部落格，底下tad543專案為目錄名稱

  • 建立完，登入該資料夾，並安裝依賴套件

  • 生成網站頁面

  • 啟用測試站

然後打開瀏覽器，輸入 http://localhost:4000 即可
